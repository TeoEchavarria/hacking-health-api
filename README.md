# 🏥 Hacking Health API

El corazón de la plataforma Hacking Health. Un backend robusto que conecta la salud del usuario a través de múltiples dispositivos y aplicaciones.

## 🎯 Visión

Hacking Health es un ecosistema integral de salud que empodera a los usuarios para tomar control de sus datos de bienestar. Esta API es el núcleo central que:

- **Sincroniza datos** desde la app móvil y el smartwatch
- **Gestiona la autenticación** segura de usuarios
- **Procesa información** de salud, alimentos y actividad física
- **Genera insights** personalizados basados en los datos del usuario

## 🔗 Conectado con

- **[hacking-health-app](../hacking-health-app/)** - La aplicación móvil que captura datos detallados de salud y nutrición
- **[hacking-health-watch-app](../hacking-health-watch-app/)** - El smartwatch que monitorea actividad y métricas en tiempo real

Juntos forman una plataforma unificada donde cada dispositivo contribuye datos que se sincronizan a través de esta API central.

## 📡 Qué Hace

Gestiona usuarios, autenticación, datos de salud, registros de alimentos, appointments, y proporciona un sistema completo de insights y análisis basado en los datos recopilados desde todos los dispositivos conectados.
